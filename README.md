# Detecção de clientes de alto risco

Projeto final da minha pós-graduação em Ciência de Dados e Machine Learning na PUC Campinas.

O estudo feito em Jupyter Notebook com a biblioteca Python pandas, scikit-learn e seaborn, avalia e compara e performance dos modelos Regressão Logística e Random Forest para análise de clientes de alto risco (maus pagadores) de um banco fictício.
