# Regressão Logística e Random Forest na detecção de clientes de alto risco

Projeto final da minha pós-graduação em Ciência de Dados e Machine Learning na PUC Campinas.

O estudo feito em Jupyter Notebook com a biblioteca Python pandas, numpy, scikit-learn, seaborn e matplotlib, avalia e compara a performance dos modelos Regressão Logística e Random Forest para detectar clientes de alto risco (maus pagadores, ou seja, aqueles que possuem alta probabilidade de causar prejuízos financeiros, cometer fraudes, realizar lavagem de dinheiro ou gerar problemas de conformidade legal/reputacional) de um banco fictício.

![image](https://github.com/user-attachments/assets/2c5d4cdd-f51f-435f-8f78-c3987e2d4e02)

Considerou-se 0 sendo os bons pagadores e 1 os maus pagadores.

O modelo de Regressão Logística apresentou desempenho ligeiramente superior ao Random Forest, principalmente ao demonstrar um maior recall para a classe de risco (1), reduzindo falsos negativos, o que é crucial para minimizar perdas financeiras.
